# Team Capacity Management - Frontend

React frontend for the Team Capacity Management API.

## Setup

1. Install Node.js dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

The app will open at http://localhost:3000

## Features

- Create sprints with team members, holidays, and vacations
- View all sprints
- Calculate and display capacity for each sprint
- Real-time capacity calculations
- Responsive UI

## API Connection

The frontend connects to the FastAPI backend at `http://127.0.0.1:8000/v1`

Make sure your backend server is running before starting the frontend.
